Semântica em HTML5

O que é Semântica?

É o significado dos elementos HTML e como eles descrevem seu conteúdo.

Exemplo: <article> para conteúdo independente, <nav> para menus

Importância no HTML5

  • Melhora a acessibilidade
  • Otimiza o SEO
  • Facilita a manutenção
  • Organiza melhor o código

Elementos Semânticos

header, nav, main, section, article, aside, footer
