Linguagem Server Side vs Client Side

Server Side (Lado do Servidor)

Executadas no servidor antes de enviar a página ao navegador.

Exemplos:

  • PHP
  • Python (Django, Flask)
  • Java
  • C# (ASP.NET)

Client Side (Lado do Cliente)

Executadas no navegador do usuário.

Exemplos:

  • JavaScript
  • TypeScript
  • HTML (marcação)
  • CSS (estilos)

Diferenças Principais

  • Server Side: Processa no servidor, acessa bancos de dados, mais seguro
  • Client Side: Roda no navegador, permite interatividade, visível ao usuário
